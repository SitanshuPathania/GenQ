# GenQ
The primary objective of the GenQ(text-to-quiz generation website) is to provide educators and
content creators with a user-friendly platform to effortlessly convert text passages into quiz
questions. By leveraging natural language processing (NLP) techniques and machine
learning algorithms, the website automates the extraction of key concepts and generates
diverse question types. The user has to just provide a passage and an answer and a
question will be automatically generated by our the algorithm of our website. Additionally,
the website aims to offer a seamless user experience, with intuitive interfaces for inputing
text passages, reviewing generated questions, and exporting quizzes in various formats.
